# 🎟️ Seat Booking System (React)

A simple and interactive **Seat Booking Web Application** built using React.
This project allows users to select available seats, view booking details, and calculate total cost dynamically.

---

## 🚀 Features

* 🪑 Display available and booked seats
* ✅ Select and deselect seats
* 🎨 Visual seat status (Available / Selected / Booked)
* 💰 Real-time total price calculation
* ⚡ Fast and responsive UI using React

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR-USERNAME/seat-booking-app.git
```

### 2️⃣ Navigate to project folder

```
cd seat-booking-app
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the application

```
npm start
```

---

## 🌐 Usage

* Click on available seats to select them
* Selected seats will change color
* View selected seats and total price in the summary section
* Booked seats cannot be selected
